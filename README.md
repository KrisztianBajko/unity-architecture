# Unity Architecture

Opinionated architecture for professional Unity development.

This repository contains the public website, handbook, documentation, roadmap, and brand assets for the Unity Architecture ecosystem.

## Current Status

- **Runtime Core** — Released as `v0.1.0`
- **Save** — Architecture design complete; implementation pending
- **Website** — Initial setup in progress

## Repository Structure

- `website/` — Astro website and developer portal
- `handbook/` — Handbook source files and composed chapters
- `documentation/` — Package and API documentation
- `branding/` — Brand identity, logos, icons, and visual guidelines
- `assets/` — Screenshots, diagrams, and downloadable media
- `roadmap/` — Public roadmap and release planning
